# Theory overview

The SGTA project is supported by a formal mathematical formulation
covering representation geometry, memory dynamics,
graph construction, and stability considerations.

The theoretical material is currently under internal revision
and is not publicly released at this stage.

The formulation includes:
- a spherical representation space,
- a synalytic memory defined by geometric regions,
- explicit update dynamics for memory parameters,
- an energy-based view of stability,
- a graph structure over concepts.

The theoretical framework is sufficient to support
the experimental findings reported in this repository.

Open theoretical questions concern rigorous stability analysis,
scaling behavior, and connections to generalization theory.

The formal material is available for private discussion if relevant.
